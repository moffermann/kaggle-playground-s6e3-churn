# Setup

## Requisitos

- Cuenta Kaggle
- Kaggle API token (`kaggle.json`) configurado
- Aceptar las reglas de la competencia en Kaggle

## Comandos utiles

Descarga de datos:

```bash
kaggle competitions download -c playground-series-s6e3 -p data/raw
```

Extraccion:

```bash
unzip data/raw/playground-series-s6e3.zip -d data/raw
```

## Inscripcion

Si, para descargar datos y enviar predicciones con tu cuenta debes:

1. Entrar a la competencia.
2. Hacer click en `Join Competition`.
3. Aceptar las reglas.

Sin eso, Kaggle normalmente bloquea el acceso a los archivos de competencia.
