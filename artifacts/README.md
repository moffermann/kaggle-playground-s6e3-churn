# Artifacts

Salida local de ejecución del baseline.

- `models/`: modelos entrenados (`.cbm`)
- `reports/`: métricas y reportes (`.json`)
- `submissions/`: archivos listos para enviar a Kaggle (`.csv`)
- `logs/`: logs opcionales de ejecución

Estos artefactos se generan por los scripts en `scripts/`.
